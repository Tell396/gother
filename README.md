<h2 align="center">Installation</h2>

__1.__ [Install Golang from official site.](https://go.dev/dl/)\
__2__. Download repository:
```
git clone https://github.com/tell396/gother && cd gother
```
__3.__ Run the program:
```
go run main.go
```
